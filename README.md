# TaskManager API

## Prerequisites

- Node.js
- npm
- mongodb

## Installation

1. Clone the repository:

```
https://github.com/bewuket0/task-manager-api
```

2. Navigate to the project directory:

```
cd task-manager-api
```

3. Install dependencies:

```
npm install
```

4. Run the project:

```
npm run server
```

the documentation is available on the : <url>/docs
